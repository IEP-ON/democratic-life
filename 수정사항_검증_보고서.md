### 4. LLM 흔적 기호(#, **) 제거

**대상 파일**: 지도안 및 학습지 HTML 파일

**수정 내용**:

#### (1) 마크다운 볼드체 (**텍스트**)
- 패턴: `**텍스트**`
- 수정: `텍스트` (단순 제거)
- 예시:
  - 수정 전: `**포트폴리오 정리 (20분)**`
  - 수정 후: `포트폴리오 정리 (20분)`

#### (2) 마크다운 제목 (###, ##, #)
- 패턴: `### 제목`, `## 제목`, `# 제목`
- 수정: `제목` (# 기호 제거)
- HTML에서는 불필요한 마크다운 문법

**수정 방법**:
```python
# remove_markdown_symbols.py 실행
# 정규표현식으로 ** 및 # 기호 제거
```

**수정 결과**:
- ✅ 지도안 39개 파일 수정
- ✅ 학습지는 마크다운 기호 없음 (확인 완료)

**검증**:
```bash
grep -r "\*\*" 지도안/*.html
# 결과: 0건 (모두 제거됨)
```

## ⚠️ 주의 사항 및 제외 항목

### 1. "▣" 기호는 유지
- "▣"는 한국 교육 현장에서 일반적으로 사용하는 활동 제목 기호
- LLM 흔적이 아님
- 예: `▣ 동기유발`, `▣ [Connect] 세계시민 개념 연결하기`

### 2. 학습지는 학습목표 표기 없음
- 학습지는 활동 중심 구성
- 학습목표는 지도안에만 명시
- 따라서 학습지는 수정 대상 아님

### 3. "핵심 내용" 표현은 유지
- 학습지의 표 제목으로 사용되는 "핵심 내용"은 유지
- "핵심개념"과는 다른 맥락
- 예: `<th>핵심 내용</th>` (학습지 표 제목)
