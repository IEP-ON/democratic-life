웹 보고서를 온라인으로 공개하려면 GitHub Pages를 활성화해야 합니다.

### 설정 방법

1. **GitHub 저장소 접속**
   ```
   https://github.com/IEP-ON/democratic-life
   ```

2. **Settings → Pages로 이동**
   - 저장소 상단 메뉴에서 "Settings" 클릭
   - 왼쪽 사이드바에서 "Pages" 클릭

3. **Source 설정**
   - **Branch**: `main` 선택
   - **Folder**: `/docs` 선택
   - "Save" 버튼 클릭

4. **배포 완료 확인**
   - 1-2분 후 자동 배포 완료
   - 주소 확인: `https://iep-on.github.io/democratic-life/`
